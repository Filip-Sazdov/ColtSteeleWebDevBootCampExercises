Arrays Problem Set

Write printReverse(), a function that takes an array as an argument and prints out the elements in the array in reverse order.

Write isUniform(), a function that takes an array as an argument and returns true if all elements in the array are identical.

Write sumArray(), a function that takes an array of numbers as an argument and returns the sum of all numbers in the array.

Write max(), a function that takes an array of numbers as an argument and returns the maximum number in the array.
